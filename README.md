# Поиск геометрической медианы

## Описание

Данный проект реализует вычисление геометрической медианы для набора точек на двумерной плоскости. Искомая точка минимизирует суммарное евклидово расстояние до всех заданных точек.

Ключевые особенности:

- использование алгоритма Вайсфельда (Weiszfeld's algorithm);
- отказ от тяжелых зависимостей (типа `scipy.optimize`) в пользу чистого `numpy` для бесшовного C/C++ портирования;
- строгий контроль памяти через contiguous-буферы.

## Подробности алгоритма

Математическое обоснование и детали метода вынесены в отдельный документ: [METHODOLOGY.md](METHODOLOGY.md).

## Установка

Установите зависимости командой:

```bash
pip install -r requirements.txt
```

Для проекта требуются только `numpy` и `matplotlib`.

## Использование

Запуск выполняется командой:

```bash
python solution.py
```

В текущей демонстрационной конфигурации исходные точки захардкожены в скрипте `solution.py`. Координаты найденной геометрической медианы выводятся в консоль, а график автоматически сохраняется в `data/output/result.png`.

## Структура проекта

- `solution.py` — точка входа и оркестрация;
- `geometric_median_math.py` — вычислительное ядро без внешних зависимостей;
- `visualizer.py` — отрисовка графика.

## Примечания по производительности

Архитектура проекта ориентирована на подход System-ready и реализована в процедурном стиле. Вычислительное ядро не использует скрытых абстракций и heavyweight-оптимизаторов, что упрощает перенос алгоритма на C/C++, в том числе для сценариев запуска на мобильных процессорах и на голом железе.
